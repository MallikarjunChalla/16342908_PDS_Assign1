# Student Performance Analysis

Welcome to the Student Performance Analysis project! This Python project aims to analyze student performance data to gain insights into factors affecting academic achievement.

## Project Overview

This project consists of two main source files:

1. **data_preparation.py**: This script handles data preparation tasks such as loading the dataset, cleaning the data, and preparing it for analysis.
   
2. **data_analysis.py**: This script performs various analyses on the cleaned dataset, including exploratory data analysis, visualization, and potentially predictive modeling.

## Project Structure

The project contains the following directories:

- **raw_data**: This folder holds the raw data CSV file.
- **clean_data**: This folder contains the cleaned data generated by the data preparation script.
- **results**: This folder contains all the results generated by the data analysis script.
