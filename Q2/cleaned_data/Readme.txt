
Welcome to the Cleaned Data directory for the Student Performance Analysis project! This directory contains the cleaned data generated after processing the raw data CSV file.

File Description
The cleaned data file contains the following columns:

gender: Gender of the student (male or female).
race/ethnicity: Race/ethnicity group the student belongs to.
parental level of education: Highest level of education attained by the student's parent or guardian.
lunch: Type of lunch received by the student (standard or free/reduced).
test preparation course: Whether the student completed a test preparation course (none or completed).
math score: Score achieved by the student in the math exam.
reading score: Score achieved by the student in the reading exam.
writing score: Score achieved by the student in the writing exam.


##Usage
This cleaned data file serves as the processed and prepared dataset for the Student Performance Analysis project. It has been cleaned and formatted to facilitate further analysis, such as exploring factors affecting student performance.
