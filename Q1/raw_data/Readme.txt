
Welcome to the Raw Data directory for the Frailty Analysis project! This directory contains the raw data CSV file used in the analysis.

File Description
The raw data file contains the following columns:

Height: Height of the participants in inches.
Weight: Weight of the participants in pounds.
Age: Age of the participants in years.
Grip Strength: Grip strength of the participants measured in kilograms.
Frailty: Qualitative attribute indicating the presence or absence of symptoms associated with frailty (N indicates absence, Y indicates presence).
##Usage
This raw data file serves as the initial input for the Frailty Analysis project.
